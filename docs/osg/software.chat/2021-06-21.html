(11:02:22) miwalls: Is there a way to find condor_ce_q jobs in squeue by job id or another identifier?
(11:22:39) matyas: The attribute that contains the slurm job ID in condor_ce_q is "GridJobId".
Do condor_ce_q -constraint RoutedJob -af ClusterId ProcId GridJobId
You'll get output that looks like
18 0 batch slurm psc-bridges2-ce2.svc.opensciencegrid.org_9619_psc-bridges2-ce2.svc.opensciencegrid.org_18.0_1624291121 slurm/20210621/1796616

that very last number (1796616) is the job ID on the SLURM side
