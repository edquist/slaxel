(13:06:25) cnatzke: Hi all, I am running a workflow using containers and I am getting jobs occasionally failing due to the error FATAL: kernel too old. Is there a requirement I can add to help reduce these failures?
(13:07:09) dweitzel: Hi Connor, yes, we have a recipe for that.
(13:07:23) dweitzel: Add to your job requirements:
Requirements = HAS_SINGULARITY == True && OSG_HOST_KERNEL_VERSION >= 31000

(13:07:35) dweitzel: The important bit is the OSG_HOST_KERNEL_VERSION
(13:07:44) dweitzel: (documented here: https://support.opensciencegrid.org/support/solutions/articles/12000024676-docker-and-singularity-containers#fatal-kernel-too-old)
(13:08:32) dweitzel: The reason I know this so well is that I recently hit the same issue.
(13:09:59) cnatzke: Perfect! Thanks so much!
