(17:39:49) kretzke: I figured out why the Kibana “discover” tab times out so often for gracc… As of 4.3 Kibana uses the _field_stats API to get the date range for every index matching the pattern, which means elasticsearch has to hit every shard when you first load the page. So that’s what’s timing out, not actually loading data. Using the legacy time-based index pattern solves that, but it’s deprecated. Monthly indices will certainly also help.
